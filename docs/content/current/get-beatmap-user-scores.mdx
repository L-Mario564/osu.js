# Get beatmap user scores

The `beatmaps.getBeatmapUserScores` method on a `Client` instance makes a `GET` request to the `beatmaps/{beatmap}/scores/users/{user}/all` endpoint and returns a user's scores on a beatmap.

### Example

```ts
const scores = await api.beatmaps.getBeatmapUserScores(1816113, 12408961, {
  query: {
    mode: 'osu'
  }
});
```

### Parameters

| Parameter | Type                                                              | Optional | Description                          |
| --------- | ----------------------------------------------------------------- | -------- | ------------------------------------ |
| beatmap   | number                                                            |          | ID of the beatmap to get scores from |
| user      | number                                                            |          | ID of the user to get scores from    |
| options   | [GetBeatmapScoresOptions](/references/get-beatmap-scores-options) | ✓        |                                      |

#### options.query

| Parameter | Type                              | Optional | Description                      |
| --------- | --------------------------------- | -------- | -------------------------------- |
| mode      | [GameMode](/references/game-mode) | ✓        | Gamemode of the scores to return |

### Returns

```ts
Promise<Score[]>;
```
