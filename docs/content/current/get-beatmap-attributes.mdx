# Get beatmap attributes

The `beatmaps.getBeatmapAttributes` method on a `Client` instance makes a `POST` request to the `beatmaps/{beatmap}/attributes` endpoint and returns a beatmap's attributes.

### Example

```ts
const scores = await api.beatmaps.getBeatmapAttributes(1816113, 'fruits');
```

### Parameters

| Parameter | Type                                                                      | Optional | Description                             |
| --------- | ------------------------------------------------------------------------- | -------- | --------------------------------------- |
| beatmap   | number                                                                    |          | ID of the beatmap to get its attributes |
| gamemode  | [Gamemode](/references/game-mode)                                         |          | Gamemode attributes to get              |
| options   | [GetBeatmapAttributesOptions](/references/get-beatmap-attributes-options) | ✓        |                                         |

#### options.body

| Parameter | Type   | Optional                 | Description |
| --------- | ------ | ------------------------ | ----------- | -------------------------------------------------------------------------------- |
| mods      | number | [Mod](/references/mod)[] | ✓           | Mods to apply (can be either the bitwise representation or an array of acronyms) |

### Returns

If `gamemode` is set to `osu`:

```ts
Promise<OsuBeatmapDifficultyAttributes>;
```

#### Types

- [OsuBeatmapDifficultyAttributes](/references/osu-beatmap-difficulty-attributes)

If `gamemode` is set to `taiko`:

```ts
Promise<TaikoBeatmapDifficultyAttributes>;
```

#### Types

- [TaikoBeatmapDifficultyAttributes](/references/taiko-beatmap-difficulty-attributes)

If `gamemode` is set to `fruits`:

```ts
Promise<FruitsBeatmapDifficultyAttributes>;
```

#### Types

- [FruitsBeatmapDifficultyAttributes](/references/fruits-beatmap-difficulty-attributes)

If `gamemode` is set to `mania`:

```ts
Promise<ManiaBeatmapDifficultyAttributes>;
```

#### Types

- [ManiaBeatmapDifficultyAttributes](/references/mania-beatmap-difficulty-attributes)

### Possible Exceptions

See the [error handling documentation](/extras/error-handling) for more information.

- `invalid_json_syntax`
- `network_error`
- `unexpected_response`
