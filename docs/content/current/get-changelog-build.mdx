# Get changelog build

The `changelog.getChangelogBuild` method on a `Client` instance makes a `GET` request to the `changelog/{stream}/{build}` endpoint and returns a changelog build.

### Example

```ts
const build = await api.changelog.getChangelogBuild('lazer', '2023.305.0');
```

### Parameters

| Parameter | Type                                            | Optional | Description        |
| --------- | ----------------------------------------------- | -------- | ------------------ |
| stream    | [ChangelogStream](/references/changelog-stream) |          | Update stream name |
| build     | string                                          |          | Build version      |

### Returns

```ts
Promise<
  Build & {
    changelog_entries: (ChangelogEntry & {
      github_user: GithubUser;
      message: string | null;
      message_html: string | null;
    })[];
    versions: BuildVersions;
  }
>;
```

#### Types

- [Build](/references/build)
- [ChangelogEntry](/references/changelog-entry)
- [GithubUser](/references/github-user)
- [BuildVersions](/references/build-versions)

### Possible Exceptions

See the [error handling documentation](/extras/error-handling) for more information.

- `invalid_json_syntax`
- `network_error`
- `unexpected_response`
