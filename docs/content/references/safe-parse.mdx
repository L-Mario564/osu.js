# SafeParse

```ts
type SafeParse<TData, TUsePolyfillResponse extends boolean = false> =
  | {
      success: true;
      data: TData;
    }
  | {
      success: false;
      response: TUsePolyfillResponse extends true ? PolyfillResponse : Response;
    };
```
